统一能力

商户号
{$user.pid}
接口地址
{$baseUrl}
能力地址说明
查询订单信息/api.php?act=order返回订单金额、实付金额、时间、状态等完整信息。
查询订单状态/api.php?act=order_status只返回支付状态和状态文本。
关闭订单/api.php?act=close仅待支付订单可关闭，关闭后释放金额占用。
查询服务端状态/api.php?act=server_status返回服务名称和服务器时间。
 
查询订单信息：
{$baseUrl}/api.php?act=order&pid={$user.pid}&key=API_KEY&out_trade_no=商户单号

查询订单状态：
{$baseUrl}/api.php?act=order_status&pid={$user.pid}&key=API_KEY&out_trade_no=商户单号

关闭订单：
{$baseUrl}/api.php?act=close&pid={$user.pid}&key=API_KEY&out_trade_no=商户单号

查询服务端状态：
{$baseUrl}/api.php?act=server_status

易支付

创建订单
{$baseUrl}/submit.php
移动端创建订单
{$baseUrl}/mapi.php
创建订单字段：
pid={$user.pid}
type=wxpay 或 alipay
out_trade_no=商户单号
money=订单金额
name=商品名称
notify_url=异步通知地址
return_url=同步跳转地址
param=透传参数
sign=MD5签名
sign_type=MD5

签名规则：
参数按 ASCII 排序，排除 sign、sign_type、空值，拼接 k=v&... 后追加 API KEY，MD5 小写。

码支付

创建订单
{$baseUrl}/creat_order/ 
创建订单字段：
id={$user.pid}
type=1 或 2
pay_id=商户单号
price=订单金额
name=商品名称
notify_url=异步通知地址
return_url=同步跳转地址
param=透传参数
sign=MD5签名

渠道映射：
type=1 微信支付
type=2 支付宝

签名规则：
md5(pay_id + param + type + price + API_KEY)

源支付

创建订单
{$baseUrl}/yuanpay/submit
移动端创建订单
{$baseUrl}/yuanpay/mapi
创建订单字段：
pid={$user.pid}
type=wxpay 或 alipay
out_trade_no=商户单号
money=订单金额
name=商品名称
notify_url=异步通知地址
return_url=同步跳转地址
param=透传参数
sign=MD5签名
sign_type=MD5

签名规则：
与易支付一致，参数按 ASCII 排序后追加 API KEY，MD5 小写。

回调参数说明

易支付 / 源支付回调：
pid=商户号
trade_no=平台订单号
out_trade_no=商户单号
type=wxpay 或 alipay
name=商品名称
money=实付金额
trade_status=TRADE_SUCCESS
sign=MD5签名
sign_type=MD5

成功响应：
success
码支付回调：
id=商户号
pay_id=商户单号
param=透传参数
type=1 或 2
price=订单金额
reallyprice=实付金额
sign=MD5签名

成功响应：
success
